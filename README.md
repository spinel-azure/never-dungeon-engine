# Never Dungeon Engine (NDE)

Never Dungeon Engine（NDE）は、ブラウザ上で動作するクラシックスタイルの疑似3Dダンジョン探索エンジンです。

10×10セルのランダムダンジョンを生成し、PC、タブレット、スマートフォンから探索できます。探索エンジンの基盤は、2026年7月22日に **Ver.1.0** として完成しました。

## 主な機能

### ダンジョン生成と探索

- Canvas 2Dとレイキャストによるクラシックな疑似3D描画
- 10×10セルのランダム迷路生成
- 全100セルへ到達可能な経路保証
- 行き止まりとループ経路を含む通路構成
- ランダムな開始地点
- 前進、後退、左右旋回のアニメーション
- 壁、マップ外、閉じた扉への衝突判定
- 通常扉、ボス部屋用扉、施錠扉の配置
- 扉の開閉アニメーション
- 上り階段と下り階段の3D表示
- 下り階段による次階層への移動
- 階層移動時の開始座標継承
- 階層ごとに変化する壁色・床色

### フロアテーマ

- 壁色6種類：デフォルト、赤、青、緑、白、黒
- 床色6種類：デフォルト、赤、青、緑、紫、白
- B2F以降は階層移動時に壁色と床色をランダム変更
- 直前の階層と同じ色を避けるテーマ選択
- 固定テーマ、ランダムテーマ、階層別指定へ対応できるテーマ管理基盤
- 緑、青白、紫の霧
- 霧の濃さと描画距離の調整

### ミニマップと探索支援

- 探索済み範囲を記録するミニマップ
- 未探索セルのノイズ表示
- 壁、扉、階段、NPC、宝箱、プレイヤー位置のマップ表示
- プレイヤーの向きを示すコンパス
- 座標と階層の表示
- 探索済み経路だけを通るオート帰還
- ダンジョンのランダム再生成
- 探索開始からの経過時間表示
- 階層移動時のラップタイム表示

### たいまつと気配

- 移動に応じて減少するたいまつゲージ
- たいまつ残量50％未満から強まる暗闇演出
- たいまつ消失時のミニマップ砂嵐表示
- 0～100で蓄積する気配ゲージ
- 通常時と暗闇で異なる気配上昇量
- 特殊イベントマスを除外した遭遇判定
- 気配最大時の仮エンカウント通知

戦闘システムは未実装のため、現在は遭遇通知のみ表示します。

### NPCイベント

- NPCデータの一元管理
- 進行不能を起こさない安全なセルへのランダム配置
- 疑似3D空間へのNPCスプライト表示
- 正面にいるNPCの気配通知
- 汎用オーバーレイによる会話イベント
- 括弧内の台詞を一文字ずつ表示するタイプライター演出
- タイプライター表示のON／OFFと速度設定
- Aボタンによる会話進行
- Bボタンによる会話終了
- 遭遇回数の記録と会話後のNPC消去

現在実装されているNPCは「みかんにゃんこ」1体です。

### 宝箱イベント

- 赤、黒、金の宝箱を各1個ランダム配置
- 階段やNPCとの重複、配置による進行不能を回避
- 疑似3D空間での宝箱表示
- Three.jsを使用した宝箱開封演出
- 宝箱の種類ごとに異なる回転、開閉、発光演出
- 開けずに立ち去った宝箱の再発見
- 種類別ミニマップアイコン

宝箱の中身は仮実装です。

### メニューとオプション

- 常設コマンドメニュー
- 簡易ステータス表示
- 2ページ構成のステータス画面
- 画面の揺れのON／OFF
- たいまつの揺らぎのON／OFF
- NPCタイプライター表示と速度設定
- SEのON／OFFと音量設定
- LocalStorageによる設定保存
- タイトル画面、favicon、Apple Touch Icon

### サウンド

- 移動、衝突、扉、階段、宝箱、メニュー操作用SE
- 階段移動と帰還完了時の連続SE
- PC向けの足音・衝突音
- モバイル／タブレットでは負荷軽減のため足音・衝突音を無効化
- SEの遅延読み込み
- 再生優先度と最大同時再生数の管理
- 非表示・ページ離脱時の全SE停止
- 将来の戦闘用SEを接続できる音声管理基盤

BGMは未実装です。

### マルチデバイス対応

- PC、タブレット、スマートフォン向けレスポンシブUI
- キーボード、マウス、タッチ操作
- タッチ端末用バーチャルスティック
- タッチ端末用A／Bボタン
- iOSブラウザのURLバーを考慮した可視領域追従
- ホーム画面からの起動を考慮したモバイル表示

## 操作方法

| 操作 | キーボード | 画面操作 |
| --- | --- | --- |
| 前進 | `↑` | バーチャルスティック上 |
| 後退 | `↓` | バーチャルスティック下 |
| 左旋回 | `←` | バーチャルスティック左 |
| 右旋回 | `→` | バーチャルスティック右 |
| 決定・調べる・会話を進める | `X` | Aボタン |
| キャンセル・メニュー表示 | `Z` | Bボタン |

## ローカルでの実行

ES Modulesを使用しているため、`file://`による直開きではなくHTTPサーバー経由で開いてください。

静的ファイルのみで構成されているため、GitHub Pagesなどの静的ホスティング環境で公開できます。
